# Mehjabin Shirin - Academic CV Website

This is a static academic CV and research profile website.

## Main files

- `index.html` - website content
- `style.css` - design and layout
- `script.js` - mobile menu, animation, gallery filtering, and lightbox
- `Mehjabin_Shirin_CV.pdf` - downloadable public CV
- `Mehjabin_Shirin_CV_2026.md` - editable text version of the CV
- `GALLERY_SETUP_STEP_BY_STEP.md` - exact instructions for adding real gallery images

## Important before publishing

1. Replace gallery placeholder SVG files with final images.
2. Add Web3Forms access key in `index.html` for the contact form.
3. Verify all award, conference, and presentation details.
4. Do not upload patient-identifiable images, private emails, phone numbers, addresses, or unapproved copyrighted material.
